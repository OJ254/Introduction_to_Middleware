# **Band Name Generator**

This is a simple Express.js web application that generates a band name based on user input.

## **Features**
- Serves a homepage with an HTML form.
- Handles form submissions and constructs a band name using user-provided values.
- Uses Express.js and Body-Parser for handling HTTP requests.

## **Installation**
### **1. Clone the repository:**
   ```sh
   git clone https://github.com/OJ254/Introduction_to_Middleware.git
   ```
### **2. Navigate to the project directory:**
   ```sh
   cd Introduction_to_Middleware
   ```
### **3. Install dependencies:**
   ```sh
   npm install
   ```

## **Usage**

1. Start the server:
   ```sh
   nodemon index4.js
   ```
2. Open your browser and go to `http://localhost:3000`.
3. Enter your street name and pet's name in the form.
4. Submit to generate your band name.
